# 현재 구현 상태

이 문서는 현재 코드 기준의 구현 범위를 정리한다.

## 구현 완료

### 설치와 실행 기반

- 설치 상태 분기
- 웹 설치 화면
- `config/config.php` 생성
- `storage/installed.lock` 생성
- core/member/admin 설치 SQL 실행
- 스키마 버전 기록
- Docker 기반 로컬 PHP 실행 래퍼

### 코어

- 요청 method/path 해석
- 설치 전/설치 후 분기
- 활성 모듈 `paths.php` 기반 action include
- action 상대 경로 검증
- 기본 home/error view
- CSRF helper
- HTML escape helper
- redirect helper
- HMAC hash helper
- 감사 로그 helper
- 예외 요약 파일 로그

### 회원

- 회원가입
- 로그인/로그아웃
- 내 계정 화면
- 비밀번호 변경
- 비밀번호 재설정 token 생성/검증/사용 처리
- 이메일 인증 token 생성/검증/사용 처리
- 회원 탈퇴 상태 처리
- 약관/개인정보 동의 기록
- 개인정보 요청 접수/목록
- 개인정보 JSON 내보내기
- 인증 로그 기록

### 관리자

- 관리자 대시보드
- 사이트 설정 조회/저장
- 모듈 목록/상태 관리
- 기본 모듈 비활성화 차단
- 회원 목록/상태 관리
- 관리자 역할 부여/회수
- 마지막 owner 회수 차단
- 감사 로그 조회
- 개인정보 요청 처리
- 개인정보 요청 JSON 내보내기
- 보관 기간 수동 정리

### 개인정보와 운영

- 감사 로그 테이블
- 개인정보 요청 테이블
- 개인정보 내보내기에서 내부 hash 제외
- 사용 완료 token 정리
- 오래된 인증 로그/감사 로그 정리
- 배포 보호 기준 문서화
- `storage/logs/error.log` 오류 기록

## 남은 작업

- 메일 발송 연동
- 프로필 선택 정보
- DB 기반 세션/강제 로그아웃
- 업데이트 실행 UI
- 파일 로그
- CSS/UI 정돈
- 서버별 배포 예시 문서
- 모듈별 개인정보 export 확장 규약

## 검증 기준

현재 구현은 다음 검증을 통과했다.

```text
전체 PHP 파일 php -l
git diff --check
설치/로그인/관리자 주요 화면 수동 HTTP 검증
회원가입/비밀번호 변경/재설정/이메일 인증/탈퇴 검증
개인정보 요청/내보내기/보관 정리 검증
```
