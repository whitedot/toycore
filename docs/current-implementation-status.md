# 현재 구현 상태

이 문서는 현재 코드 기준의 구현 범위를 정리한다.

## 구현 완료

### 설치와 실행 기반

- 설치 상태 분기
- 웹 설치 화면
- `config/config.php` 생성
- `storage/installed.lock` 생성
- core/member/admin 설치 SQL 실행
- 스키마 버전 기록
- Docker 기반 로컬 PHP 실행 래퍼

### 코어

- 요청 method/path 해석
- 설치 전/설치 후 분기
- 현재 locale 결정
- 기본 번역 helper
- 사이트/모듈 설정 조회 helper
- 요청 단위 설정 조회 메모리 캐시
- 활성 모듈 `paths.php` 기반 action include
- action 상대 경로 검증
- 기본 home/error view
- 공통 CSS 파일
- CSRF helper
- HTML escape helper
- redirect helper
- 기본 mail helper
- HMAC hash helper
- 감사 로그 helper
- 예외 요약 파일 로그

### 회원

- 회원가입
- 로그인/로그아웃
- 회원가입 이메일 인증 메일 발송 요청
- 내 계정 화면
- 비밀번호 변경
- 비밀번호 재설정 token 생성/검증/사용 처리
- 비밀번호 재설정 메일 발송 요청
- 이메일 인증 token 생성/검증/사용 처리
- DB 기반 로그인 세션 기록
- 선택 프로필 저장/조회
- 회원 탈퇴 상태 처리
- 탈퇴 시 선택 프로필 삭제
- 약관/개인정보 동의 기록
- 개인정보 요청 접수/목록
- 개인정보 JSON 내보내기
- 개인정보 JSON 내보내기 프로필 포함
- 개인정보 JSON 내보내기 세션 이력 포함
- 모듈별 개인정보 JSON 내보내기 확장
- 인증 로그 기록

### 관리자

- 관리자 대시보드
- 사이트 설정 조회/저장
- 사이트 설정 항목 조회/저장/삭제
- 모듈 목록/상태 관리
- 기본 모듈 비활성화 차단
- 회원 목록/상태 관리
- 회원 활성 세션 수 조회
- 회원 세션 강제 폐기
- 관리자 역할 부여/회수
- 마지막 owner 회수 차단
- 감사 로그 조회
- 개인정보 요청 처리
- 개인정보 요청 JSON 내보내기
- 보관 기간 수동 정리
- 업데이트 확인/실행

### 개인정보와 운영

- 감사 로그 테이블
- 개인정보 요청 테이블
- 개인정보 내보내기에서 내부 hash 제외
- 사용 완료 token 정리
- 만료/폐기 세션 정리
- 오래된 인증 로그/감사 로그 정리
- 배포 보호 기준 문서화
- `storage/logs/error.log` 오류 기록
- 모듈 업데이트 SQL 적용

## 남은 작업

- 현재 문서화된 1차 구현 목표는 모두 코드 또는 문서로 반영됨

## 검증 기준

현재 구현은 다음 검증을 통과했다.

```text
전체 PHP 파일 php -l
git diff --check
설치/로그인/관리자 주요 화면 수동 HTTP 검증
회원가입/비밀번호 변경/재설정/이메일 인증/탈퇴 검증
개인정보 요청/내보내기/보관 정리 검증
관리자 업데이트 화면 검증
DB 세션 기록/폐기 검증
공통 CSS 정적 파일 응답 검증
```
