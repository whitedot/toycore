UI Guide Dashboard

사용 맥락 구분

이 항목은 기존 UI-KIT 분류를 유지하면서 Base, Admin, Public, Skin 맥락으로 나누어 확인합니다.

Base
공통 원형

common.css의 토큰과 기본 컴포넌트 클래스를 우선 확인합니다.

Admin
관리자 화면

목록, 설정, 필터, 반복 작업처럼 밀도 높은 운영 화면에서 쓰는 조합입니다.

Public
사용자 화면

회원, 콘텐츠, 공개 페이지처럼 읽기 흐름과 화면 맥락이 중요한 조합입니다.

Skin
스킨 예시

스킨은 자기 영역 안에서만 최종 표현을 조정하고 공통 원형을 직접 흔들지 않습니다.

테이블 맥락별 예시

Base
공통 테이블

기본 .table과 .table-wrapper 구조를 사용합니다.

Admin
관리자 목록

필터, 상태 배지, 행 단위 액션을 함께 배치합니다.

대량 작업과 반복 확인을 우선합니다.
Public
사용자 목록

표가 꼭 필요할 때만 사용하고, 콘텐츠 흐름이 중요하면 카드 목록을 우선합니다.

Skin
스킨 목록

스킨은 행 간격, 강조색, 보조 텍스트 표현만 자기 범위에서 조정합니다.
