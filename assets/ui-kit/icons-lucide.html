UI Guide Dashboard

사용 맥락 구분

이 항목은 기존 UI-KIT 분류를 유지하면서 Base, Admin, Public, Skin 맥락으로 나누어 확인합니다.

Base
공통 원형

common.css의 토큰과 기본 컴포넌트 클래스를 우선 확인합니다.

Admin
관리자 화면

목록, 설정, 필터, 반복 작업처럼 밀도 높은 운영 화면에서 쓰는 조합입니다.

Public
사용자 화면

회원, 콘텐츠, 공개 페이지처럼 읽기 흐름과 화면 맥락이 중요한 조합입니다.

Skin
스킨 예시

스킨은 자기 영역 안에서만 최종 표현을 조정하고 공통 원형을 직접 흔들지 않습니다.

개요 (Overview)

Lucide는 웹 및 앱 개발을 위한 깨끗하고 확장 가능한 SVG 아이콘의 오픈 소스 라이브러리로, 쉬운 통합과 맞춤 설정을 제공합니다.
