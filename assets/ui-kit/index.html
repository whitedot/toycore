Saanraan UI Guide

Saanraan UI 모음

Saanraan 관리자 화면과 사용자 화면에서 사용하는 UI를 같은 컴포넌트 분류 안에서 확인하는 개발자 참고 화면입니다. 버튼, 카드, 알림, 배지, 모달, 드롭다운, 탭, 폼, 테이블, 아이콘 예시는 기존 UI-KIT 항목을 유지합니다.

각 항목 안에서는 Base, Admin, Public, Skin 맥락을 구분해 확인하고, 실제 화면에서는 common.css, admin-ui.css, public-ui.css, 스킨 CSS를 역할별로 분리해 사용합니다.

레이어 기준

Base
공통 원형

common.css의 토큰과 기본 컴포넌트 클래스를 기준으로 삼습니다.

Admin
관리자 UI

admin-ui.css에서 운영 화면의 밀도와 반복 작업 패턴을 확장합니다.

Public
사용자 UI

public-ui.css에서 공개 화면, 회원 화면, 콘텐츠 흐름을 확장합니다.

Skin
스킨

스킨은 자기 범위 안에서 최종 표현을 조정하고 Base/Admin/Public 원형을 직접 오염시키지 않습니다.
